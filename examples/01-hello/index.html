01 - Hello World

The simplest Aft example: display a value from data.

{"greeting": "Hello Universe"}

What's happening?

  • <aft-form> creates a reactive context
  • <aft-data> provides JSON data
  • aft-text="greeting" displays the value of greeting

The Code

<aft-form>
  <aft-data>{"greeting": "Hello Universe"}</aft-data>

  <p><span aft-text="greeting"></span></p>
</aft-form>
