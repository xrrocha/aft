05 - Iteration with Tables

Use aft-each on a <tr> to build dynamic tables.

{ "products": [ {"name": "Widget A", "price": 25.00, "qty": 2}, {"name": "Widget B", "price": 15.50, "qty": 1}, {"name": "Widget C", "price": 42.00, "qty": 3}, {"name": "Widget D", "price": 8.75, "qty": 5} ] }
# Product Price Qty Subtotal
$ $
Grand Total: $

What's happening?

  • aft-each="products" repeats the <tr> for each item
  • aft-index-as="i" provides the 0-based index
  • aft-ref="product.qty" enables two-way binding inside iteration
  • The grand total updates automatically when any quantity changes
