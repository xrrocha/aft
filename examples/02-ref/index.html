02 - Two-Way Binding

Use aft-ref to bind an input to data. Changes flow both ways.

{ "planet": { "greeting": "Hello Earth" } }

Current value:

What's happening?

  • aft-ref="planet.greeting" binds the input to nested data
  • Typing in the input updates the data
  • The aft-text output updates automatically

Path Expressions

Paths use JavaScript property access: planet.greeting, items[0].name, etc.
